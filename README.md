# AIFFEL_quest_rs

해당 레포지토리는 아이펠 리서치 과정 "7-2 프로젝트 : GitHub 레포지토리 구성하기"를 구현한 것입니다!!!  
구현한 내용은 다음과 같습니다.

- AIFFEL 과정 중 진행하는 프로젝트, 피어리뷰를 관리하는 Repository입니다.
- 기본적인 틀을 제공하며, 파일이나 폴더를 수정/업로드하여 사용합니다.
- 주의:Flutter 프로젝트는 **프로젝트 폴더**를 업로드합니다.
